Past state of being can be expressed by adding `だった` to the Noun or Na-Adjective.
### Example
```
学生だった
Translation:
Is not Student

Kanji used:
学生 = がくせい, meaning Student
```

---
### Negative Past State of being
To negate the state of being, you add `じゃなかった` instead of `じゃない`.
### Example
```
学生じゃなかった
Translation:
Was not Student.

Kanji used:
学生 = がくせい, meaning Student
```
